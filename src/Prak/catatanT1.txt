2. Menggunakan array untuk menyimpan elemen stack.
3. Method push digunakan untuk menambahkan elemen ke dalam stack, sedangkan method pop digunakan untuk menghapus elemen dari stack.
4. Method reverseString digunakan untuk membalikkan urutan string. Setiap karakter dari string ditambahkan ke dalam stack menggunakan method push. Kemudian, setiap karakter dihapus dari stack menggunakan method pop dan ditambahkan ke dalam string hasil balik.
5. Di dalam method main, user diminta untuk memasukkan sebuah string. Kemudian, method reverseString dipanggil untuk membalikkan urutan string, dan hasilnya ditampilkan pada layar.
